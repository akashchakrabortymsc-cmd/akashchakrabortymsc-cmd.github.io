Why I Am Building volesti-rs — March 18, 2026

No crate on crates.io implements Ball Walk, Hit-and-Run, or Billiard Walk for convex polytopes. I searched. The gap is real.

The GeomScale papers — Bachelard, Chalkis, Fisikopoulos, Tsigaridas (AISTATS 2023) — solved the problem of sampling from disconnected spherical geometry for portfolio analysis. The C++ implementation exists. The R bindings exist. The Rust port does not.

I started building volesti-rs before the GSoC proposal window opened because I did not want to wait. Current status: 18 tests passing, 15µs/sample at d=50 (release profile), PSRF R̂ = 1.03 on H-cube10.

Project repo: github.com/akashchakrabortymsc-cmd/Volesti_Rust
